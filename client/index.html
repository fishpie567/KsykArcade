Setup guide

  1. Install Node.js 18+ on your server or local machine.
  2. Duplicate the .env.example file (created in the repository root) to .env and fill in your credentials.
  3. Run npm install inside the server folder if you want to add extra packages. The project works without additional dependencies.
  4. Start the backend with node server/index.js. The same server will host this frontend.
  5. Open http://localhost:4000 in your browser to access the arcade.
  6. Configure PayPal sandbox credentials and update the script tag at the top of this page with your client ID.
  7. Optional: Enable Google sign-in by entering your GOOGLE_CLIENT_ID value in .env and the Google script configuration in app.js.
